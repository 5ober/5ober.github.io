Linux 配置静态IP

Rocky 9.0 NetworkManager 配置文件位置（ens160网卡名称）

修改网卡配置文件

vim /etc/NetworkManager/system-connections/ens160.nmconnection

[ipv4]

//添加ip/子网掩码,网关

address1=192.168.1.30/24,192.168.1.254

//添加dns地址（多个用;隔开）

dns=8.8.8.8;114.114.114.114;

may-fail=false

method=manual

重新加载配置文件

nmcli connection load /etc/NetworkManager/system-connections/ens160.nmconnectioni

激活配置文件

nmcli connection up /etc/NetworkManager/system-connections/ens160.nmconnection
